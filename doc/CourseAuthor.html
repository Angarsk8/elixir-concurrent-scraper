    Courses Scrapper v0.0.1 CourseAuthor

    Provides a set of functions to crawl the data of a course's author. It brings the ability to build a struct with the needed information about the author.

    Example

    iex> author_tree = <some_nested_tree_object>
<some_nested_tree_object>
iex> CourseAuthor.build_author_struct(author_tree)
%CourseAuthor{
  name: "Rob Percival",
  twitter: "https://twitter.com/techedrob",
  facebook: "https://www.facebook.com/rpcodestars"
  googleplus: "none",
  linkedin: "none",
  youtube: "https://www.youtube.com/user/robpercival"
}

    Functions

    __struct__()

    Defines the fixed structure of an author of a course

    build_author_struct(author_tree)

    Build a struct that identifies an author of a course based on the HTML document tree passed as argument

    get_author_name(author_tree, query)

    Parses an HTML document tree that represents an author and find the name based on a CSS query string

    get_contact_link(author_tree, keyword)

    Parses an HTML document tree that represents an author and pattern match against a CSS query string to find his social media contact link
